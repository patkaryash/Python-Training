# Coding-Prashant-JHA

A day-wise Python learning repository with small practice programs, short notes, and beginner-friendly examples. Each folder focuses on one stage of the learning journey so topics stay easy to revise.

## Learning Journey

| Day | Main Focus |
| --- | --- |
| `DAY1` | Python basics, data types, operators, type casting, conditions, and beginner logic building |
| `DAY2` | Lists, slicing, tuples, loops, aliasing, and string-based practice |
| `DAY3` | Functions, arguments, dictionaries, and nested loop exercises |
| `DAY4` | Predefined string methods, exception handling, file handling, and CSV work |
| `DAY5` | OOP basics: classes, objects, constructors, inheritance, polymorphism, and CRUD practice |
| `DAY6` | Abstraction, encapsulation, stacks, and a few logic-based problem-solving tasks |
| `DAY7` | Time complexity, searching, stacks, queues, and linked list basics |
| `DAY8` | Matrix input practice and row-wise operations using nested lists |
