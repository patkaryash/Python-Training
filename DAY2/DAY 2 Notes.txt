DAY 2 NOTES
============

Strings and Collections
-----------------------

1. Extracting a substring from a string is called slicing.
2. Python provides collection data types for storing multiple values.
3. A list stores data in order and is:
   - heterogeneous
   - growable
   - mutable
4. A tuple is immutable.

Aliasing
--------

Aliasing in a list means assigning one variable reference to another variable.

List vs Tuple
-------------

- Use a tuple when the values are not expected to change.
- Use a list when the values may change in the future.

`zip()` Function
----------------

The `zip()` function can be used to take multiple iterables together and access their values in parallel.
