DAY 3 NOTES
============

Functions
---------

1. A function is a self-contained block of code that can be executed separately.
2. There are two main types of functions:
   - built-in functions
   - user-defined functions
3. Built-in functions are already available in Python.
4. User-defined functions are created by the programmer according to program requirements.

Types of Arguments
------------------

The main types of arguments used in functions are:

1. Positional arguments
2. Keyword arguments
3. Default arguments
4. Variable-length arguments

Dictionary
----------

A dictionary represents data in key-value pairs.

String Trimming Functions
-------------------------

1. `rstrip()` removes spaces from the right-hand side.
2. `lstrip()` removes spaces from the left-hand side.
3. `strip()` removes spaces from both sides.
