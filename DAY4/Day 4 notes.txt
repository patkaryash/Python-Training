DAY 4 NOTES
============

Predefined Functions
--------------------

Python provides many predefined functions and string methods that help us perform common tasks quickly.

Exception Handling
------------------

Exception handling is used to manage runtime errors so that the rest of the program can continue to run normally.

Main idea:
- the error part is handled separately
- the normal flow of execution can continue

Important rule:
- the default `except` block should be written last, otherwise it can cause a syntax or logic problem

File Handling
-------------

File handling is used to read data from files, write data to files, and manage stored information using Python programs.
