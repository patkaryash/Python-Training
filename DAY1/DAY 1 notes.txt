DAY 1 NOTES
============

Python Basics
-------------

1. Python is called a dynamically typed language because the Python interpreter decides the value type at runtime.
2. All fundamental data types are immutable.
3. Converting a value from one data type to another is called type casting.
4. The `input()` function accepts data as a string by default.
5. Python has reserved keywords that are built in to the language syntax.

Operators
---------

1. Identity operators are used when we want to compare addresses or object identity.
2. Membership operators are used to check whether an object is present inside a collection.

Examples of collections:
- list
- tuple
- set
- dictionary
- string
