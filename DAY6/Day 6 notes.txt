DAY 6 NOTES
============

Abstract Class
--------------

1. A class that contains one or more abstract methods is called an abstract class.
2. An abstract method has a declaration but does not have an implementation.
3. We use abstract classes while designing large functional units.
4. We also use them when we want to provide a common interface for different implementations.

Encapsulation
-------------

Encapsulation is the process of protecting data and functionality inside a single unit.

Data Structures
---------------

Data structures are different ways of organizing data on a computer so that it can be used effectively.

Algorithm
---------

An algorithm is a step-by-step procedure to solve a problem.

Tower of Hanoi
--------------

Rules:

1. We can move only one disk at a time.
2. We have to pick the upper disk from any one peg.
3. We have to place it on top of another peg.
4. We cannot place a larger disk on top of a smaller disk.
